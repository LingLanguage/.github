# LingLang 玲珑语言
## 作者碎碎念
1. 第一次写编译器，总得从经历中成长，没写成也可以当作下一次的经验积累  
2. 大项目中，权限控制很重要，这是当前大多数语言缺失的，这是我想实现的关键一点  
3. 多线程经常打乱架构，我想独立出一套多线程控制的方法  
4. 二进制操作应当支持得更友好，比如int8[] ToBytes(T obj)和T FromBytes<T>(int8[] src)
5. 不想取代什么，只想写个像我这样的程序员喜欢的语言
